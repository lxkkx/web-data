# 🚀 AI Task Management & Geolocation Web App

A premium, state-of-the-art web application converted from the original Android frontend. This project features a sophisticated dark theme, AI-powered productivity insights, and seamless task management.

---

## ✨ Features
- **Premium Aesthetics**: Curated dark mode design with glassmorphism and smooth transitions.
- **AI Integration**: Predictive deadline risks, smart rescheduling, and AI-powered task analysis.
- **Team Collaboration**: Shared spaces, team chat, and performance leaderboards.
- **Geolocation Support**: Map interface for tracking tasks with geofenced location reminders.
- **Micro-Animations**: Extensive use of `framer-motion` for a responsive and alive interface.

---

## 🛠️ Tech Stack
- **Core**: React 19 (Vite)
- **Styling**: Tailwind CSS v4 (with custom `@tailwindcss/postcss`)
- **Animations**: Framer Motion & Lottie React
- **Icons**: Lucide React
- **Routing**: React Router DOM v7

---

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### 2. Installation
Open your terminal and run the following commands:
```bash
# Navigate to the project directory
cd c:/Users/sodas/OneDrive/Desktop/website/web-frontend

# Install dependencies
npm install
```

### 3. Running the App
Start the development server:
```bash
npm run dev
```
The app will be available at: **[http://localhost:5173](http://localhost:5173)**

## 📂 Project Structure
- `/src/pages/auth`: Login, Registration, Password Reset, and Verification flow.
- `/src/pages/dashboard`: Home, Profile, Notifications, and Analytics.
- `/src/pages/tasks`: Task listings, Detailed views, and Creation/Edit forms.
- `/src/pages/team`: Collaboration tools, Team Chat, and Analysis.
- `/src/pages/map`: Geolocation and map-based task tracking.

---

## 💡 Usage Notes
- **Lottie Animations**: Located on the Splash and Onboarding screens.
- **Navigation**: Use the bottom navigation bar to switch between Home, Tasks, Map, Team, and Profile.
- **AI Interaction**: Access the AI Assistant from the Home screen or the "Insights" section in Profile.
